# VideoPlayer
使用ijkVideoView做底层,默认使用surfaceView， 

功能: 1 滑动改变(音量、进度、亮度)
      2 双击暂停/播放
      3 后台秒恢复
      4 自己旋转(比较简陋，还需要优化）
      5 toolBar(横屏隐藏，竖屏（播放的时候隐藏，暂停的时候始终显示）)
      6 锁屏逻辑
      7 支持 自定义UI

         5 弹幕
         6 网络监听
